Convex hull project info

07/16/2017

The solution consists of 4 projects: 1) ConvexHullConsole - main project - contains the logic and presentation in console; 2) ConvexHull - WPF-version - for visual display; 3) ConvexHullConsole.Tests - tests for ConvexHullConsole; 4) ConvexHullDocumentation - project for generating documentation.

Can improve: 1) Image loading and points find further improvement: use of specialized libraries in the processing of PNG images, work with the file directly, using assembler; 2) Convex hull scan further improvement: to move some calculations out from the cycles, to find (for example here: https://designengrlab.github.io/MIConvexHull/) or to write other implementations of the algorithm, compare and choose more suitable algorithm for the task; 3) Improve test coverage, add random tests, tests on large data; 4) Conduct code profiling and work on bottlenecks

ConvexHull section presents more detail description.
